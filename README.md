# BeepLocator
~$5 BLE Locator with Android App

## What is this?

This is a roughly $5-$10 BLE Beacon/Keys locator that is open source, easy to build and most importantly; Fun!

## Status

This project is still heavily in development.

### v1 Motherboard design
V1 had a design flaw in that you really can't easily solder onto the battery pad on the back. A bodge wire to override battery support, and it works! ...kinda.
